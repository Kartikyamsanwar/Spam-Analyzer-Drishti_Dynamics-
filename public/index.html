📌 About the Project

The Spam Analyzer is a futuristic AI-based web application that detects spam across text, images, and videos.

  • ✅ Text Analyzer: Uses GPT to classify spam or not spam.
  • ✅ Image Analyzer: Uses DALL·E/Vision APIs to detect misleading or harmful content.
  • ✅ Video Analyzer: Uses Sora (prototype integration) to analyze video context.

With a sleek UI and multi-modal analysis, our solution demonstrates how AI can protect users from spam in every form and build a safer, trustworthy digital space.
